Домашнее задание 1: внешняя сортировка большого файла

Описание программы
==================

Проект содержит программу для сортировки большого файла с целыми числами.

Основная программа работает с бинарным файлом, в котором подряд записаны 32-битные целые числа без заголовка. Размер исходного файла может превышать объем оперативной памяти, поэтому файл не загружается в память целиком.

Программа создает новый отсортированный файл в той же директории, где находится исходный файл.

Основной файл программы:

external_binary_sort_u32.py

Эта программа принимает два параметра:

1. имя бинарного файла;
2. максимальное количество чисел, которые разрешается одновременно загрузить в память.

Программа выполняет внешнюю сортировку: исходный файл делится на части, каждая часть сортируется отдельно, затем отсортированные части объединяются в один итоговый файл.

Сортировка частей выполняется параллельно с использованием всех доступных процессорных ядер.


Дополнительная программа
========================

Также в проекте есть вспомогательная программа:

txt_to_binary_then_sort.py

Она нужна для случая, если исходный файл дан в текстовом формате, например random_numbers.txt, где на каждой строке записано одно целое число.

Эта программа сначала преобразует txt-файл в бинарный файл, а затем запускает основную программу сортировки.

Основной программой решения является:

external_binary_sort_u32.py


Файлы проекта
=============

external_binary_sort_u32.py
    Основная программа для сортировки бинарного файла с 32-битными целыми числами.

txt_to_binary_then_sort.py
    Вспомогательная программа для преобразования txt-файла в бинарный формат и последующей сортировки.

README.txt
    Описание программы и инструкция по запуску.


Запуск основной программы
=========================

Если исходный файл уже находится в бинарном формате, нужно запускать:

python external_binary_sort_u32.py <binary_file> <max_numbers_in_memory>

Пример:

python external_binary_sort_u32.py random_numbers_binary.bin 1000000

Здесь:

random_numbers_binary.bin
    имя исходного бинарного файла;

1000000
    максимальное количество чисел, которые можно одновременно загрузить в память.

После выполнения программа создаст новый файл в той же директории.

Например, если входной файл называется:

random_numbers_binary.bin

то результат будет называться:

random_numbers_binary_sorted.bin


Запуск для txt-файла
====================

Если исходный файл находится в текстовом формате, где на каждой строке записано одно число, нужно запускать вспомогательную программу:

python txt_to_binary_then_sort.py <txt_file> <max_numbers_in_memory>

Пример:

python txt_to_binary_then_sort.py random_numbers.txt 1000000

После выполнения будут созданы два файла:

random_numbers_binary.bin
    бинарная версия исходного txt-файла;

random_numbers_binary_sorted.bin
    отсортированный бинарный файл.


Пример запуска в PyCharm
========================

Для запуска программы в PyCharm нужно открыть Run/Debug Configurations.

Для основной программы нужно указать:

Script path:
external_binary_sort_u32.py

Script parameters:
random_numbers_binary.bin 1000000

Для вспомогательной программы нужно указать:

Script path:
txt_to_binary_then_sort.py

Script parameters:
random_numbers.txt 1000000

Если файл находится не в папке проекта, нужно указать полный путь к файлу.

Например:

"D:\sort_test\random_numbers.txt" 1000000
